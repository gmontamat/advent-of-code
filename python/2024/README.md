# Advent of Code 2024

** Not following advent calendar **

## Usage

We use self-contained [`uv`](https://docs.astral.sh/uv/) scripts:

1. [Install `uv`](https://docs.astral.sh/uv/getting-started/installation/)
2. Run solutions passing input file: `./solution1.py <file>` (for example `./solution2.py input`)
